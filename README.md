# Noteblock

Simple JavaScript text editor with multiple tabs.
The text is saved in Chrome local storage.
File download and mailto functions.

## Install

- `npm install`

### Production build

- `npm run build`

### Development build

- `npm start`

## Firefox

- set manifest version to `2`
- set `actions` to `browser_action`
